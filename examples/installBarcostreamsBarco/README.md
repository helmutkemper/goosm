# Install MongoDB (no ephemeral)

## English

This example installs **Barco** on ports **2950**, **2951** and **2952** using **golang** and **docker**.

#### Requirements:

[Docker Desktop](https://www.docker.com/products/docker-desktop/) installed

#### How to run the example:

```shell
  make build
```
